# 你写这篇文章什么目的
这篇文章将告诉你你不该在 GroupServer 中做什么，你应该做什么，你可以怎样为服务器做贡献，并指导你成为一名优秀的服务器玩家。

# 你不应该这么做
## 和其他玩家对骂
我认为加入 GS 的玩家们都应该是素质高的——至少不会乱骂人。如果你是属于乱骂人的“高”素质玩家，你可以考虑转服。

**你不应该像这样做**
```
[主世界] [普通人] Player: 你好
[主世界] [普通人] SunXiaoChuan: 你好傻逼，你要干啥？
[主世界] [普通人] Player: 有话好好说，别骂人
[主世界] [普通人] SunXiaoChuan: 不服就滚，臭傻逼，你妈死了

**玩家心里：什么**服务器，全都无缘无故骂人，退了退了**

[-] Player

```

**这会造成什么后果**
* 新玩家发现这个服务器的玩家素质都很“高”，退了退了，呆不下去
* 有些玩家的心理素质比较差，可能骂骂就会被逼退服

**遇到这种玩家怎么办**
向腐竹或其他管理举办，如果证据充足，最高可获 4000 金币奖励。腐竹 QQ 744198767，邮箱 creeper938@outlook.com。

**你该怎么做**
做一个素质高的玩家，至少要不乱骂人。遇到上面这样的玩家，积极向管理举报。

## 偷其他玩家的东西、乱拆东西
服务器玩家的家、私有财产被偷的事件常有发生。几乎每个月，腐竹都能收到玩家向腐竹举报自己家被熊的事件。

**你不应该像这样做**
* 发现别人的东西很值钱，我直接给拆了
* 在别人放告示牌警告“不要乱拆”的情况下，我依然无视
* 腐竹是什么东西？我直接给你东西全偷了

**这会造成什么后果**
* 玩家们发现自己好不容易搞来的东西就这样被偷了，心态炸了，退服
* **作案者会被封禁 5 ~ 60 天**

**遇到这种玩家怎么办**
向腐竹或其他管理举办，如果证据充足，最高可获 20000 金币奖励或 200000 金币补偿。如果损失较大，腐竹会给你的家回档。

**你该怎么做**
不要乱拆其他玩家的东西、房子。不要偷其他玩家的东西。如果你想要他的东西，请和他尝试通过友好的方式进行索要，而不是直接偷。

## 利用服务器漏洞
利用服务器漏洞刷物品，甚至造成服务器严重卡顿、关服，都是严重违反服务器规则的，会被严厉制裁。

**你不应该像这样做**
* 我找到个刷物品的 Bug！来来来，我告诉你们怎么用，不要让管理发现哦
* 这个 Bug 能让服务器关服，哈哈，等腐竹搞了个我不满意的东西我就让他尝尝关服的滋味

**这会造成什么后果**
* 修不好，插件直接被删了，永远不给你加回去
* 刷的东西太多了，造成经济、发展失衡
* 服务器可能会因为修漏洞维护个好几天，让你好几天都玩不到服务器
* **作案者会被封禁 30 天，甚至永久**

**遇到这种玩家怎么办**
向腐竹举办（管理管不了），如果证据确凿，最高可获 300000 金币奖励。如果作案者情节恶劣，可能会被开。

**你该怎么做**
发现 Bug 就应该立即向腐竹举报！能获得 6000 ~ 20000 的金币奖励

## 开大纪
开挂是作弊行为，作弊可耻。服务器不是 2B2T 这种的无政府服务器，因此本服务器不欢迎作弊者。请不要抱有侥幸心理，认为管理抓不到你。

**你不应该像这样做**
* 我开个杀戮光环刷怪，应该没事吧？反正腐竹也不在
* 这老逼登装备太强了，我注入个 Vapu 干他吧

**这会造成什么后果**
* 破坏公平性
* **作弊者会被封禁 7 ~ 365 天**

**遇到这种玩家怎么办**
向管理举报。建议附带截图。如果证据充足，最高可获 8000 金币奖励。

**你该怎么做**
不要保有侥幸心理，认为管理发现不到你。不要试图在服务器内开挂。

## 攻击、压测服务器
和腐竹、管理或玩家发生争执，应该用和平方式去解决，而不是去攻击服务器，这不会对你有任何好处，对服务器和你都有坏处。

**你不应该像这样做**
* 妈的，腐竹这个更新真傻*。我要压测服务器！！！
* 什么垃圾服务器，做的太他妈烂了，我去花钱找人攻击了

**这会造成什么后果**
* 服务器使用旋律云的面板服，如果你 DDoS 服务器，相当于 DDoS 了旋律云，进而影响使用旋律云的绝大部分服务器
* 压测服务器会导致服务器的玩家一直进不去服务器或让服务器处于卡顿状态，进而导致玩家流失

# 你该怎么做
## 为新手资助
向新手资助装备物品能给他们留下一个好的印象，有几率留住他们。
**注意: 不要给他们一些很强大的装备，也不要上来就给下界合金，否则会让他们失去目标，然后就会导致他觉得没意思，退服了。**

**如何辨认新手**
仅从游戏名字辨认这个玩家是不是新人并不行。你得从他的行为特征、聊天的讲话风格、以及 IP 归属地去判断他是不是某个人的小号，专门装新人骗装备的。

## 积极反馈 Bug、为服务器发展提建议
如果你发现了 Bug，那么一定要向腐竹反馈，可以获得金币奖励。

**如何判断这是不是 Bug**
多次通过同样的方法，看看能否复现。看看是不是正常的特性，别把挖草方块会掉泥土当 Bug 了。

另外，服务器现在发展已经逐渐到瓶颈了。希望你能为服务器的发展提一些建议，建议如果被采纳可以获得金币奖励。

## 做一名不违反服规的好玩家
违反服规，就会像上面的章节一样受到惩罚。希望你能不违反服务器规则，做一名优秀玩家，发现违反服规的玩家要积极举报。

## 和其他玩家友好相处
你需要和其他玩家友好相处才能融入社区。
互相尊重，互相理解，互相合作才能更好地融入社区。你不能总是做一些他不想让你干的事情，尽量不让他失望。
不要拆其他玩家的房子，不要去偷，否则你会成为服务器里最臭名昭著的玩家。

## 如果你有钱没处花的话...
不如赞助服务器？服务器是公益服，没有氪金项目所以不能盈利。为了服务器的长久发展，希望你能赞助服务器。赞助方式见服务器指南。
如果你手头里资金紧张，那还是不要赞助了。

# 正确对待，而不是过激反应
## 对于更新
如果你对某次更新感到不满，说明这可能侵犯到了你的利益。如果你确实认为是这样，你应该去想想整个服务器现在的状况。我最近的所有更新基本都是在试图维持平衡性，如果平衡性被破坏了，就会导致发展速度过快，进而造成玩家提前毕业，造成大量玩家结业退服。
如果你有建议，可以向腐竹反馈，而不是直接上来就骂，那样对你没有好处。

## 如果你家被熊
你需要冷静对待，及时向腐竹举报。如果你现在很慌，你需要冷静下来，否则腐竹、管理也会很着急，你越慌，你退服的几率就会越大。退服后，你过不了多久就会后悔。
你需要保持一个良好的心态去玩服务器。不要因为家被熊就不玩了。服务器还有很多发展方向供你选择。

# 题外话
## 为什么你的服务器玩家流失这么严重

玩家流失，是一个很正常的现象，尤其是在小服务器。玩家流失有很多原因，我们将以玩家的视角，从多种角度来进行分析。

{% note warning %}
**注意，本章节并不适用于所有生存类型的服务器。且这些内容是贴合我的服务器目前状况来写的。如果您有意见，不要上来就骂，在评论区向我指正哪里有错误，我会及时更改。**
{% endnote %}

### 从玩家的角度进行分析

#### 发现了一个服务器

**注意：下面的剧情均为虚构。**

我是一个 MC 玩家，我很喜欢联机，因为我觉得单人太没意思了。有一天，我在 B 站上找到了一个服务器，看上去很吸引人。正当我准备进入服务器时，却发现这个服务器的进入流程是如此复杂...... 要搞外置登录什么的，还得去配置启动器。对于我这种懒人来说，这也太不友好了。

{% note info %}
**一、进入服务器的流程对玩家不友好**
许多服务器为了防熊，搞了一套白名单制或者外置登录。其中外置登录是对玩家最不友好的，大部分玩家发现 “又要注册账号又要配置启动器的，真麻烦，不搞了不搞了”，通常就会放弃这个服务器。
**建议:** 简化进入服务器的流程。尽量不要搞外置登录，会导致玩家流失很严重。外置登录对手机玩家来说是更不友好的。建议改为白名单制。如果非要搞外置登录，建议将进入教程写的详细点。
{% endnote %}

#### 建了一个家
终于，我进入了这个服务器。进入服务器后，我发现这里的玩家素质都好低啊。动不动就骂人..... 这服务器对新人也太不友好了吧！

我忍着其他玩家们的互相谩骂和指责，在服务器里建造了一个小家。建完后，我就去睡觉了。第二天我放学回家，我发现我的房子居然被毁了！！！

{% note info %}
**二、服务器社区风气不好，素质低下，熊孩子多**
服务器的玩家社区决定着一个服务器能否良好发展。如果熊孩子多，其他正常玩家的房子天天被搞，就会导致正常玩家的心态爆炸，从而退服。`“老子建了好几天的房子被毁了！心态炸了！！！玩不下去了！！！！！”`
如果玩家的素质都很低，经常骂人，就会给新人留下一个不好的印象。这会让玩家不敢在服务器里面做一些醒目的事，因为他们觉得这样做可能会让那些骂人的玩家感到不满。一些心理防线低的玩家可能会直接退服。
**建议:** 把服务器规则搞得严一点，尤其是对于熊别人房子的。当然也不要太严，否则照样会造成玩家的流失。加强服务器的管理，多招几个能长期在线的管理，让他们根据服务器规则封禁熊孩子。加入敏感词系统，屏蔽一些骂人的词。
{% endnote %}

#### 服务器逐渐变凉
上次那件家被毁的事情发生后，我向腐竹举报。腐竹赔偿了我金币并且帮我回档了，我很开心，决定要留在这个服务器。

{% note %}
积极处理玩家举办的事件、反馈等等可以增加玩家粘性！会让玩家觉得这个服务器管理很负责任
{% endnote %}

然而，这个服务器我玩着玩着，就逐渐变凉了，从以前的每天 40 人在线，逐渐变成了 25 人，15 人，5 人，3 人......

我喜欢联机。这个服务器要是没人的话，那和单机有什么区别？？？那我还不如直接去玩单人游戏了。

{% note info %}
**三、服务器在线玩家少是导致玩家流失的重要因素**
大多数选择玩服务器的 MC 玩家都是因为觉得单人游戏很孤独，没有人陪着玩，玩不下去。或者是喜欢人多的社区。如果服务器的玩家都逐渐退服，不玩服务器了，那么剩下的玩家们就会觉得没意思，人太少了，不知道该干些什么。
如果人多起来，玩家们就有目标了，合作起来一起挖矿、钓鱼、收菜、搞建筑、肝东西之类的。所以为了能留住玩家，就得先让服务器人多起来。
玩家少的主要一大因素就是服务器发展速度太快了，玩家们已经毕业了，该退服了。或者，新玩家发现服务器已经发展到快毕业了，就会失去目标，不知道该干什么。
**建议:** 多举办活动（活动的奖励最好不要让玩家觉得太垃圾），增加玩法（不要无脑加），开新周目（绝大多数玩家都喜欢开荒）。
{% endnote %}

#### 大佬送了我一个无尽套! 但我并不喜欢
有一天，一位大佬请求传送到了我的身边。他送了我一个无尽套，说这个很强，穿上直接无敌！于是我穿上了，然后我就感觉变得很乏味，失去了挑战性，觉得很没意思，就萌生了退服的想法。

{% note info %}
**四、玩家发展速度过快，导致玩家失去挑战目标**
玩家们的发展速度越是快，越是一种不好的迹象，这意味着你可能很快就需要开新周目了。因为当一个玩家在 3 天内就肝出了一个无尽套，直接无敌，秒杀一切，你想想他还会有什么挑战呢？他会在之后很快就觉得很乏味，失去目标，退服。~~无敌是多么寂寞。~~
**建议:** 增加服务器难度，在服务器刚开荒时就加入能拖慢发展速度的插件（例如动态经济之类的插件）。加入能让玩家长期发展（可以一直肝）的插件，例如 mcMMO、粘液科技（粘液千万不要无脑加附属，血的教训）。加入会增加难度插件，当然也不要太变态了，否则玩家流失更严重。拖慢玩家发展速度，让服务器的难度更高一些，其实是好的，不要去在意玩家们的谩骂，当他们适应之后，很快就会继续玩下去的。（当然后期就别想了）另外建议尽量别让其他玩家给新人资助一些变态装备。
**注意:** 如果玩家已经发展到后期了，你再去尝试拖慢他们的发展速度，就已经晚了。有经济插件的服务器，在玩家发展到后期经济系统就会跟不存在一样，玩家们个个都是百万富翁，买一箱子下界合金块都可以，你无论加什么动态商店、动态经济的插件都没用，他们都能做到一小时内收入百万，你的努力都是白费的，如果你删了会让玩家有变态收入的玩法，还会倒逼玩家退服。你总不可能去回收他们的变态装备，对吧？那样会直接把玩家逼到退服。
{% endnote %}

#### 管理都好不负责任！
我收回我之前夸腐竹态度好的话！现在服务器的腐竹和管理都好不负责任啊。我举报一个把我一个基地毁了的人，管理连看都不看！还说 “服务器有熊孩子都是正常现象，你忍忍吧”，搞得我都想要退服，抹黑这个服务器了。

{% note info %}
**五、管理坐视不管，会让服务器风气变得更糟，让玩家失去对服务器的信心**
知道 2B2T 为什么那么乱吗？因为它是个没有规则的“无政府服务器”，任何玩家都可以随便开纪。如果一个服务器的管理（包括腐竹）对服务器里的任何事件都坐视不管，那么服务器的规则就会逐渐化为空气，玩家们都会摇身变成熊孩子，造成其他正常玩家的退服，对服务器失去信心。之前我服务器某个玩家的家被炸了，我立即就去给他还原+送那个熊他家的人封号大礼包，如果我不去处理，这名玩家可能已经退服了。
**建议:** 管理不是摆设。优先招负责任的、有责任心的管理，对整天摆烂的管理撤职。对于服务器内玩家之间发生的争执、举报，一定要及时处理，并给玩家一个好的处理结果。
{% endnote %}

#### 腐竹你还活着吗？！
玩着玩着，我已经玩了这个服务器半年了。但是自从我入坑服务器以来，我就好像没见过服务器有过更新欸。腐竹求求你赶紧复活给我们新增几个玩法吧！我都玩腻了

**三十天后**

诶呀，腐竹怎么还不更新。再不更我真的要退服了，所有玩法我都玩过了，越玩越没意思。赶紧加个新玩法吧！！！

{% note info %}
**六、勤更新，新增玩法才能吸引玩家**
如果你只在服务器初期加了几个玩法插件，然后就不更了，这就会让玩家越玩越腻，没有新鲜感，久而久之就会造成玩家流失。因此保持服务器每月都会有更新，且更新的内容不摆烂才能吸引到玩家。
**建议:** 让服务器每个月都至少有一次更新，就像我的服务器一样。建议每个月都加一个玩法插件（不要加太多），这样才能让玩家有新鲜感。
{% endnote %}

#### 这是育碧的服务器吗？
服务器怎么 Bug 越来越多！跟育碧一样！多到根本玩不下去，腐竹你快来修一下吧！这样的服务器玩着太痛苦了

{% note info %}
**七、不要让服务器变成模拟山羊**
如果你的服务器 Bug 非常多，会让玩家感到恶心，玩不下去，从而造成玩家流失，而且还会有玩家疯狂刷物品导致发展速度过快的风险。
之前我做了个模组整合，由于加了一堆兼容性比较差、Bug 较多、优化的模组，整合包的模组多到比模拟山羊的 Bug 还多，直接给我低血压治好了。
**建议:** 尽量不要让服务器出现任何 Bug。多让玩家反馈 Bug，并给他们反馈的奖励。当然自己也需要排查服务器有无 Bug。
{% endnote %}

#### 你加了啥呀，咋都不告诉我们
腐竹我听说这回你更新了，但是你为什么连更新的内容都不告诉我们，你这样跟没更新有什么区别？？？而且你的指南也是写的一摊糊涂，我都找不到菜单的打开方法。

{% note info %}
**八、把服务器指南写的详细、简洁明了一点，并且写上更新日志**
如果你加了个飞行功能，但却没有在明显的位置告诉玩家怎么开启，并且你还在服务器的宣传语上写上了 "免费飞行" ，这会让玩家认为你虚假宣传，蹭热度吸引玩家。
如果你加了个全球市场，但没在菜单中加入打开的快捷方式，就会让玩家以为你没加这个东西，让玩家以为你依然没更新服务器。
如果你服务器指南写的很烂，会让玩家找一个功能怎么用找半天。这些也是有几率造成玩家流失的。

**建议:** 勤写更新日志，让玩家知道你更了啥。加的插件如果能通过菜单打开它的 GUI，那么就在菜单里加入这个快捷方式，让玩家更方便的打开。把服务器指南写好点，写的简洁明了，别写废话，让玩家更快的找到一个功能的用法。
{% endnote %}

**九、尽量别搞反向更新**
如果换做是你，你肝了好几个月的东西突然就被删了，你会满意吗？因此，在加入一个新东西前，必须先考虑它的平衡性、特性，如果你等到玩家们都肝出来了，发现平衡性贼烂再去删，那么玩家们都会把你骂死，造成严重的玩家流失。

**建议:** 在加入插件前，先看看它的平衡性怎么样，别到加了再去看。对于粘液科技，更不要去无脑加附属。

### 其他
玩家流失的因素可能不止以上这几点。如果能让你的服务器能够火起来，需要在开荒时就做宣传，如果等到发展到后期了再做宣传，就已经晚了，此时做宣传玩家流失特别严重。如果要加玩法，必须质量要高，质量低的玩法可能会让你都嫌弃。你也可以选择加个邀请插件，可以让玩家邀请其他玩家来获得奖励。一些玩家如果认为你的服务器没什么好玩的，但服务器玩法其实已经很多了，就说明要么是你的文档写得不够好，要么是你菜单做的不好，要么是你的玩法他觉得不好玩。
通常，寒暑假时服务器日活玩家应该是最多的。如果你此时服务器很冷清，且又不是寒暑假，不要灰心，等到寒暑假时再看看。如果人数依然很少，请考虑开个新周目。如果服务器一直是一个周目，是吸引不到玩家的。如果要开新周目，记住最好要等到服务器所有玩家基本已经发展到后期了再开，并且要提醒玩家 "我们要开新周目啦！"，最好还要把旧周目的存档公开。**不要因为 Bug 之类的就直接开新周目，不然玩家全都退服。**
你的服务器 MOTD、宣传语也必须要吸引人，宣传语写的不要太晦涩难懂。建议这样写 "1.xx.x 生存服务器 | 粘液科技10+附属、超多玩法、免费飞行"。另外建议在 MCBBS 上宣传你的服务器，那里流量最大。
如果你的服务器玩家氛围比较好，在线的玩家在 10 人左右，也可以去 B 站发你玩家的沙雕日常，或者整蛊玩家的视频，标题最好能吸引人。
为了能让玩家有成就感，让玩家能玩下去，建议加入成就插件（如 AACH）并细心的配置。

### 为什么我的服务器玩家流失这么严重
在看完上面的章节后，你可能会问我 "为啥你的服务器玩家流失这么严重"。
根源其实是因为当初创立这个服务器本身就是朋友之间开着玩的，没想着要搞大。因此前期几乎没做任何宣传，或者说根本没做。
当等到服务器开始在 B 站上发宣传视频时，服务器已经发展到中期了。但此时进来的玩家依然不少，但是等到几个月后，这些玩家基本都退服了。
到了后期，我继续做宣传。此时服务器已经几乎没有人了，日活玩家根本不存在。此时服务器已经很难挖到人了，只有几个玩家加入并留了下去，且都在3个月左右转为不活跃状态。

目前服务器玩家最快可在2个月内毕业，毕业后大多选择退服，可以看出平衡性很不好，玩家发展速度太快了。但目前我也无能为力，除非开新周目，但是绝大多数玩家应该都不会同意。
另外一点就是现在服务器的风气也差，新来的玩家看到服务器风气这么差都溜了，如果要大整顿，反而会让一些老玩家退服，还可能屁用没有。
现在服务器已经逐渐发展到瓶颈了，MCBBS 插件版上有关玩法的插件在服务器里基本都有了，如果以后要更新基本也是修修 Bug、调整下难度，或者根本没有更新。
服务器学生也很多，估计等到中考、高考、寒暑假以后在线的**可能**会多**一点**。
